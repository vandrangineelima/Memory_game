# 🎮 Memory Card Game

---

## 📌 Project Description
The **Memory Card Game** is an interactive browser-based game developed using **HTML, CSS, and JavaScript**.  
The objective of the game is to match all pairs of cards in the least number of moves and shortest possible time.

This project is designed to improve front-end web development skills while creating a fun and engaging user experience.

---

## 🛠️ Technologies Used
- **HTML**
- **CSS**
- **JavaScript**

---

## ✨ Features
- 🎯 **Multiple Difficulty Levels**
  - Easy (4 cards)
  - Medium (8 cards)
  - Hard (16 cards)
  - Extreme (32 cards)

- ⏱️ **Timer**
  - Tracks the total time taken to complete the game

- 🔢 **Moves Counter**
  - Displays the number of moves made by the player

- 🏆 **Best Time Tracking**
  - Saves and shows the best completion time

- 🔄 **Reset Game Button**
  - Allows the player to restart the game anytime

- 🌙 **Dark Mode**
  - Toggle between light mode and dark mode for better user experience

- 🎉 **Winning Message**
  - Displays a success message when all card pairs are matched
